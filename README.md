# «Витина дорога в счастливое будущее»

Персональный дневник восстановления и свободной жизни.

## Возможности

- PIN-доступ к личному дневнику;
- на главном экране можно начать только **сегодняшний** день;
- фраза по умолчанию: «Это мой первый день как я бросил пить»;
- в течение текущего дня можно добавлять любое количество мыслей;
- каждая запись сохраняется с точным временем;
- архив дней в календаре;
- прошлые дни открываются нажатием на дату;
- статистика: дни, текущая серия, количество мыслей;
- мотивирующая цитата меняется каждые 10 минут;
- адаптивная вёрстка для телефона и компьютера;
- уважение `prefers-reduced-motion`;
- SQLite + production WSGI через Waitress.

## Локальный запуск Windows

```bat
.venv\Scripts\python.exe -m pytest tests -q
start_windows.bat
```

Открыть: `http://127.0.0.1:8787`

Секреты находятся в `.env` и не должны попадать в git/архив. Для нового компьютера скопируй `.env.example` в `.env` и задай свой код доступа.

## Публичный доступ

Временный HTTPS-туннель можно запустить отдельно:

```bat
npx localtunnel --port 8787 --local-host 127.0.0.1
```

Бесплатный туннель временный: URL может измениться, а процесс должен оставаться запущенным. Для постоянной ссылки нужен домен и размещение на VPS/Render/Railway/Cloudflare Tunnel с аккаунтом.

## Безопасность

Это личный дневник, поэтому общий URL не означает открытые записи: доступ к данным закрыт PIN-сессией. Не публикуй PIN вместе со ссылкой. Для реального постоянного размещения необходимо добавить HTTPS-only cookie, резервное копирование SQLite, постоянный секрет и отдельный домен.
